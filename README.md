# Country of [Dealskoo](https://www.dealskoo.com)

Country management package of [Dealskoo/admin](https://github.com/dealskoo/admin)

```base
$ composer require dealskoo\country
```

### Publish vendor

```base 
$ php artisan vendor:publish --tag=public
```
